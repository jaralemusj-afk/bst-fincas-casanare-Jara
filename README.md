instrucciones para compilar y ejecutar:

1.Crear una carpeta para el proyecto con nombre BST.

2.Guardar las tres clases dentro de esa carpeta:
* NodoBST.java
* ArbolBST.java
* Main.java

3.Verificar que cada archivo tenga el mismo nombre de la clase pública correspondiente.

4.Abrir la terminal o CMD.

5.Entrar a la carpeta del proyecto

6.El programa mostrará el menú del Árbol Binario de Búsqueda y permitirá:
* Insertar fincas
* Buscar fincas
* Mostrar recorrido PreOrden
* Mostrar recorrido InOrden
* Mostrar recorrido PostOrden
* Salir del programa
